# Somerton Man Evidence Database

Mission 2 scope: evidence acquisition and source cataloguing only. These registers are not narrative chapters and do not evaluate case hypotheses.

Status vocabulary:

- Verified: directly supported by an identified source in the source register.
- Unverified: recorded as a lead or claim but not yet checked against a reliable source.
- Conflicting: sources disagree or a source reports uncertainty.
- Unknown: no reliable information recorded yet.
- Pending verification: source located or identified, but the underlying record still needs direct retrieval/review.

Core registers:

- `source_register.csv`
- `evidence_register.csv`
- `person_register.csv`
- `location_register.csv`
- `timeline_database.csv`
- `intelligence_gap_register.csv`
- `hypothesis_register.csv`
- `image_catalogue.csv`
- `research_question_register.csv`
- `decision_log.csv`

Method notes:

- The attached PDF is catalogued as an internal prior-assessment source, not as primary evidence.
- Repository-level source IDs are permanent within this project. Do not renumber existing IDs; append new records.
- Every future claim should cite one or more `SRC-` IDs and, where relevant, `EV-`, `PER-`, `LOC-`, and `EVENT-` IDs.
