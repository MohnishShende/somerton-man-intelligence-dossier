# Context Assessment

Mission 10 has been converted from a standalone report into project-native intelligence assets in this directory. The source report remains contextual background and should not be cited as primary evidence.

## Use Rule

Context can explain plausibility, capability and institutional limits. It must not be used as proof of a case-specific event unless a direct source connects the context to the Somerton file.
