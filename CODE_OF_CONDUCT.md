# Code of Conduct

This project is a serious historical and forensic research archive.

Participants are expected to:

- Treat victims, relatives, witnesses, and researchers with respect
- Avoid harassment, sensationalism, and defamatory speculation
- Distinguish evidence from interpretation
- Accept correction when stronger evidence appears
- Preserve uncertainty where the record is incomplete

The maintainer may remove contributions that are abusive, defamatory, misleading, or inconsistent with the research ethics policy.
