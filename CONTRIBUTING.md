# Contributing

Contributions are welcome when they improve evidence quality, source transparency, reproducibility, or editorial clarity.

## Contribution Types

- Primary-source identification or transcription
- Source metadata correction
- Verification of an existing claim
- Identification of conflicting evidence
- Bibliographic correction
- LaTeX, build, or publication improvements
- Editorial clarity that does not alter evidentiary meaning

## Evidence Rules

- Do not convert a claim into an established fact without a source record.
- Preserve conflicting evidence.
- Label inference as inference.
- Use neutral language.
- Provide archive identifiers, page numbers, image numbers, URLs, access dates, or repository call numbers whenever possible.

## Pull Requests

Pull requests should explain:

- What changed
- Which claims, sources, or files are affected
- Whether the change adds, verifies, disputes, or removes a claim
- What uncertainty remains
