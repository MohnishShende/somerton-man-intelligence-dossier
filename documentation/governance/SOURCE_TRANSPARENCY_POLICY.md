# Source Transparency Policy

The dossier should make evidence traceable without violating copyright, privacy, archive rules, or ethical limits.

## Required Metadata

- Source ID
- Title or description
- Creator or institution
- Date
- Repository or publication
- Locator
- Access date
- Rights or restriction note
- Reliability note

## Restricted Sources

If a source cannot be reproduced publicly, the source log should still describe how it was located and what restrictions apply.

## Quotation

Quote only what is necessary. Prefer paraphrase with precise citation when copyright or archive restrictions apply.
