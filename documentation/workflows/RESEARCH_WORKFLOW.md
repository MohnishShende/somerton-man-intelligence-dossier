# Research Workflow

## Source Intake

For every source, create or update a source-log record with:

- Source ID
- Source type
- Repository or publication
- Creator or institution
- Date created
- Date accessed
- Locator details
- Rights or access restrictions
- Reliability notes
- Related claim IDs

## Source Classes

- Primary sources: official records, inquest material, police records, coronial documents, original photographs, contemporaneous correspondence.
- Secondary sources: books, academic articles, documentaries, later interviews, specialist reconstructions.
- Tertiary or informal sources: Wikipedia, forums, unsourced timelines, speculative media.
- Internal analysis: prior project notes, drafts, hypotheses, and the attached 2025 assessment.

## Handling Prior Analysis

Prior analysis is useful for extracting questions and claims. It is not authority. Its claims should enter the register with status `unverified` and source class `internal_prior_assessment`.

## Outstanding Questions

Questions belong in `research/questions/`. Each question should include:

- Question ID
- Related claim IDs
- Why the question matters
- Sources already checked
- Next source targets
- Current status

## Notes

Research notes should be dated, source-linked, and clearly marked as observation, inference, or question.
